# 建筑快速估价工具 - 产品需求文档

> 目标：做一个人人都能用的建筑造价估算工具，输入几个参数，30秒出结果。
> 定位：免费、开源、手机端随手用，不依赖专业造价知识。

---

## 一、核心计算公式

```
总造价 = [Σ(材料用量系数 × 当月单价) + 人工费系数 + 机械费系数] × 建筑面积

全造价 = 土建造价 + 水电费 + 暖通费 + 装修费 + 管理费利润
```

**老板提供的基准数据（土建）：**
- 目前土建造价：1200-1500元/m²（不含水电暖）

## 五、App/网页原型功能清单

### 5.1 核心功能

1. **建筑类型选择** - 菜单：住宅/厂房/场地/办公
2. **结构类型选择** - 菜单：砖混/混凝土框架/钢结构/框架-剪力墙
3. **层数选择** - 菜单：按建筑类型动态显示
4. **建筑面积输入** - 数字输入，单位m²
5. **城市选择** - 用于拉取当地材料价格
6. **一键估算** - 输出土建总造价 + 各项分解
7. **材料价格查看** - 当月各城市主要材料价格列表

### 5.2 估算结果展示

```
估算结果（无锡市，2025年4月）

建筑类型：住宅 - 高层（19层以上）
结构类型：框架-剪力墙
建筑面积：10,000 m²

土建造价：1,350 元/m² × 10,000 m² = 1,350万元
水电安装：170 元/m² × 10,000 m² = 170万元
暖通：100 元/m² × 10,000 m² = 100万元
装修（简装）：200 元/m² × 10,000 m² = 200万元
管理费+利润(12%)：约 219万元

估算总造价：约 2,039万元
单方造价：约 2,039 元/m²
```

## 六、技术方案

### 6.1 优先方案：网页版（响应式）

- HTML5 + CSS + JavaScript
- 可打包为 PWA（渐进式网页应用）
- 手机浏览器直接用，体验等同App
- 部署：Vercel（免费）或 GitHub Pages

### 6.2 后续方案：微信小程序

- 用户量大后考虑
- 上线快，无需应用商店审核

### 6.3 数据存储

- 前端：LocalStorage 缓存材料价格
- 后端（可选）：简单的 JSON 文件存储系数表
