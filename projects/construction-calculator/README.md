# 建筑快速估价工具

> 目标：做一个人人都能用的建筑造价估算工具，输入几个参数，30秒出结果。
> 定位：免费、开源、手机端随手用，不依赖专业造价知识。

---

## 目录结构

```
construction-calculator/
├── README.md           # 本文件
├── specs/              # 产品需求、设计文档
│   └── PRDs.md
├── raw-data/           # 爬取的原始数据（待清洗）
│   └── raw-data-20260411.md
├── cleaned-data/       # 清洗后的结构化数据
│   └── README.md       # 清洗规范
├── specs/              # 技术规格、系数表、参数定义
├── code/               # 代码（网页/App）
│   └── README.md
└── notes/              # 会议记录、临时笔记
    └── README.md
```

---

## 工作流程

1. **爬取** → raw-data/（原始网页、PDF片段、搜索结果）
2. **清洗** → cleaned-data/（老板提供真实数据，覆盖/替换原始值）
3. **结构化** → specs/（系数表JSON、参数定义）
4. **开发** → code/（网页版/App）
