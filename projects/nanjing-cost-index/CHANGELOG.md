| 内容 | 说明 |

## 二、各版本差异对照

### v2.3（最早，2026-04-11 21:42）
**用途**：南京官方造价指数校准版
**特点**：
- 年份指数基于南京造价指数（仅2021-2026，缺2020）
- 地区系数简化版：县城0.9 / 地级市1.0 / 二线1.1 / **一线1.2**（无具体城市）
- 年份指数：`2021年:0.88 → 2022年:0.94 → 2023年:1.00 → 2024年:1.05 → 2025年:1.10 → 2026年:1.15`
- 参数名：`year` / `material`

---

### v3.0（2026-04-11 21:50）
**用途**：南京官方材料年度系数版
**特点**：
- 改用**材料综合系数**（钢筋+水泥+混凝土+黄砂+碎石+木材加权）
- 年份：2020-2026 全部覆盖
- 材料系数：`2020年:0.84 → 2021年:1.20 → 2022年:1.13 → 2023年:1.00 → 2024年:1.06 → 2025年:1.10 → 2026年:1.13`
  - ✅ 2020年=0.84 因新冠疫情偏低，2021年=1.20 反弹属正常（老板确认）
- 参数名：`yearMaterial` / `materialWave`

---

### V1.2（2026-04-11 22:04）
**用途**：加入上海+苏南五城版
**特点**：
- 在v3.0基础上扩城市
- 新增具体城市地区系数：上海1.28 / 南京1.10 / 苏州1.15 / 无锡1.12 / 常州1.09 / 地级市1.00 / 县城0.90
- ✅ 城市系数老板已确认可用
- 参数名：`yearMaterial` / `materialWave`

---

### V1.4（2026-04-11 22:26）✅ 当前最终版
**用途**：上海官方建设工程造价指数版
**特点**：
- **核心数据替换**：yearMaterial → yearIndex（造价指数，非材料系数）
- 上海官方造价指数（上海市建筑建材业市场管理总站发布）：
  | 年份 | 指数 | 文号 |
  |------|------|------|
  | 2020年 | 0.91 | 沪建管信〔2021〕1号 |
  | 2021年 | 1.17 | 沪建管信〔2022〕3号 |
  | 2022年 | 1.12 | 沪建管信〔2023〕2号 |
  | 2023年 | 1.00 | 基准年 |
  | 2024年 | 1.05 | 沪建管信〔2025〕2号 |
  | 2025年 | 1.08 | 官方预测 |
  | 2026年 | 1.11 | 官方预测 |
  - ✅ 2020年=0.91 因疫情偏低属正常
  - ✅ 2025/2026 预测值老板已确认可用
- 参数名改：`materialWave` → **`priceWave`**
- 参数名改：`yearMaterial` → **`yearIndex`**

## 2026-04-12 回退记录

### 回退原因
豆包 SDK V2.1 数据造假：
- 5城市案例数据：3个严重失实（南京偏差-17%，常州偏差-27%）
- "2025年三季度/四季度/2026年一季度"指数：无官方依据（江苏人工价格指数基期为"下半年"，非季度）
- "管理费7.2%+利润5.6%"：无案例支撑
- yearIndex 全套季度数据：系豆包凭空编造

### V1.4 恢复内容
- 基准：2023年 = 1.00
- yearIndex：2020年0.91 / 2021年1.17 / 2022年1.12 / 2023年1.00 / 2024年1.05 / 2025年1.08 / 2026年1.11
- 年份选项：2020~2026（去掉季度选项）
- 去掉管理费/利润拆分（保持含税综合系数）
- 去掉"案例核对"功能

### 官方数据现状
- 上海造价指数：页面可查（ciac.zjw.sh.gov.cn），附件需登录
- 南京造价指数：2019Q3~2022Q2有Excel存档（基期2007年H2=100）
- 江苏人工价格指数：2026H1 = 1.00（基期2025年H2=1.00）
- 2022年后南京官方造价指数（季度）：无公开下载渠道
