# 小白启动检查清单

> 每次新 session 开始，第一件事：逐项确认完成 ✅

## 1. 必读记忆（Obsidian）
- [ ] `.memory-manifest.json` 已读（索引文件）
- [ ] `boot_memories`（10条启动必读）已读
- [ ] 今天 + 过去 3-4 天 memory 文件已读

## 2. 必读基础文件
- [ ] `MEMORY.md` 已读
- [ ] `SOUL.md` 已读
- [ ] `USER.md` 已读
- [ ] `AI_Instructions.md` 已读（Obsidian）
- [ ] `Obsidian使用规范.md` 已读（Obsidian）

## 3. learnings 文件（如果存在）
- [ ] `LEARNINGS.md` 已读
- [ ] `ERRORS.md` 已读
- [ ] `FEATURE_REQUESTS.md` 已读

## 4. Session 结束写记忆（仅结束时做）
- [ ] 新决策/配置变更 → 写入 manifest + memory/
- [ ] 踩坑/被纠正 → 写入 LEARNINGS.md / ERRORS.md
- [ ] 老板新偏好 → 评估是否加入 boot_memories

## 核心原则
- **禁止**依赖 automemory 或任何定时任务来记录 session 内容
- **必须**当下实时写入，不靠事后补
- 每次新 session 重新走一遍清单，不要跳步
